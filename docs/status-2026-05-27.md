## Feature details

### Settings
- Settings → Reading → "Daily reading goal"
- Type: Off / Minutes per day / Chapters per day
- Amount: number input (min 1, max 500)
- Defaults: Off / 20

### Library screen chip
- Appears between the streak chip and chapters chip when goal is active
- Format: `🎯 0/20 min` or `🎯 2/3 ch`
- Contains a 44px inline progress bar that fills proportionally
- When `progress >= goalAmount`: icon → ✅, chip border + text + bar turn green (`--good`)
- Chip has `title` tooltip showing the raw numbers

### Reading time tracking
- `_readSessionStart` is set to `Date.now()` each time the chapter view opens
- `_flushReadingTime()` is called from `setView()` whenever the user navigates away from the chapter view
- Elapsed minutes are accumulated in `readingMinToday` IDB setting
- `readingMinDate` guards against cross-midnight accumulation (resets to 0 when date changes)
- Sessions under 1 minute are ignored; daily total is capped at 480 minutes (8 h)

### Chapters goal
- Counts `progress` store records where `p.date === todayStr()`
- Each `markChapterComplete` call writes a record with today's date → automatically counted
