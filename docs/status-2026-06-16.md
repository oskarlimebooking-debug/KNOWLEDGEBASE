### What was built today

**Phase N42 — Reading streak freeze token** (~65 lines added):
Users can now "freeze" one missed day per week so the streak doesn't break. A 🧊 button appears in the streak chip when the user has an active streak but hasn't read today. Clicking it protects today's streak for free (1 freeze per 7-day rolling window). If today is already frozen, a dimmed 🧊 badge shows instead.

### Feature details

- **Button visibility**: 🧊 button only shown when streak > 0, today not already read, today not already frozen, and no freeze used in the last 7 days
- **Frozen badge**: after freezing, button replaced with a dimmed `🧊` badge (no re-freezing)
- **`calcStreak()` is now freeze-aware**: frozen dates are merged into the progress date set, so the streak count includes frozen days correctly
- **7-day rolling window**: `_canFreeze` checks for any frozen date within the past 7 days — 1 freeze token per week
- **14-day cleanup**: `_freezeStreak` prunes the `streakFrozenDates` array to the last 14 days on every use
- **Toast feedback**: `showToast('Streak frozen for today 🧊', 'success')` confirms the action
