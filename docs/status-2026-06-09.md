## Feature details

### N36 — Session summary banner

- **Element**: `position: fixed; bottom: 24px; left: 50%; transform: translateX(-50%)` — centered at bottom, z-index 9000
- **Appearance**: panel background, 14px border-radius, slide-up animation via CSS transform transition (250ms)
- **Content**: "📖 Session · X min · ~Y,000 words  tap to dismiss"
- **Session logic**:
  - `_n36SessionStart` — set when entering chapter view from book view; not reset on chapter-to-chapter nav
  - `_n36SessionChapters` — Set of visited chapter IDs; each chapter's `wordCount` added once
  - Reset to null/0 on back button (regardless of whether banner is shown)
  - Skip if elapsed < 30 s
- **Dismiss**: `click` handler on element + `setTimeout(4500)` both call `hide()`
- **Multi-chapter**: navigating Next/Prev accumulates words without resetting start time
