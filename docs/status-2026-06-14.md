## Feature details

### N43 — Chapter completion keyboard shortcut

- **Trigger**: `m` or `M` key while in chapter view and no input is focused
- **First press**: marks complete via `markChapterComplete(chapterId, bookId)`, sets `#ch-complete` text to "✓ Completed", shows "Marked complete ✓" success toast
- **Subsequent presses**: shows "Already completed ✓" toast (idempotent)
- **No navigation**: unlike the `#ch-complete` button (which navigates to next chapter), the keyboard shortcut stays in the current chapter
- **Guard**: `App._chapterCtx` null-check; `_isInputFocused()` guard already exists in the surrounding block
- **Cheatsheet**: row added under "Chapter reading" section
