# Headway — Status 2026-05-11

## Current phase: N53 ✅ COMPLETED TODAY

### What was built today

**Phase N53 — Library book progress badge** (~15 lines):

Added a small completion-% label next to the progress bar on every book card in the library grid. Unstarted books (0%) show no badge; in-progress books show a dimmed `N%` label; finished books (100%) show the % in accent color + bold.

**Root cause / motivation**: The library grid already had a thin 4px progress bar indicating completion %, but no numeric label. Users couldn't tell at a glance whether a bar was 20% or 80% full.

**Implementation — two changes**:

1. **CSS** (after `.book-card .progress > i`): Added `.prog-row` flex wrapper (flex, center-aligned, 6px gap, margin-top:8px), moved `margin-top` off `.progress` itself, and added `.book-pct-badge` (10px, `var(--text-dim)`) + `.book-pct-badge.done` (accent color, bold).

2. **JS `renderLibrary`**: Added `pctBadge` const — empty string when pct===0, otherwise `<span class="book-pct-badge${pct===100?' done':''}">N%</span>`. Changed the HTML from `<div class="progress">…</div>` to `<div class="prog-row"><div class="progress">…</div>${pctBadge}</div>`.
