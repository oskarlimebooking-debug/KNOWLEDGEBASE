## Feature details

### Tag filter bar in Deck Manager
- Appears only when at least one card has a tag
- "All" chip (active when no filter) + one chip per unique tag
- Each tag chip shows how many cards are **due today** in parentheses (e.g. `vocab (3)`)
- Clicking a chip activates/deactivates that tag filter
- When a filter is active, a "▶ Review 'tag'" button appears on the right — clicking it closes the manager and starts a tag-scoped review session

### Tag editing
- Deck Manager edit form gains a Tags input field
- Comma-separated entry (e.g. `vocab, formula, concept`)
- Parsed by `_parseTags`: trims, lowercases, deduplicates, truncates each tag to 30 chars
- Saved as `tags: string[]` on the card record

### Tag chips on cards
- Displayed as coloured pills below the card back text
- Colour is deterministic from tag name (8-colour pastel palette, hash-based)
- Clicking a chip in filter bar sets that tag as the deck filter

### Tag-filtered review session
- `openReviewSession('vocab')` pulls only due cards tagged "vocab"
- Header shows a yellow badge `🏷 vocab`
- Toast on empty says `No "vocab" cards due today` instead of the standard message
- Entry point: "▶ Review 'tag'" button in the tag filter bar (no change to normal 🗂 button)
