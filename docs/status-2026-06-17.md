# Headway — Status 2026-06-17

## Current phase: N45 ✅ COMPLETED TODAY

### What was built today

**Phase N45 — Smart card sort in deck manager** (~35 lines added):
A sort `<select>` now appears in the deck manager toolbar between the search bar and the Import button. Users can choose from three sort modes:
- **Newest** (default) — cards ordered by `createdAt` descending (newest created first)
- **Most struggling** — by `hardCount` descending (cards answered Hard most often appear first)
- **A → Z** — alphabetical by front text within each book group

Sort resets to "Newest" every time the deck manager is opened.

### Feature details

- **Sort persists within session**: changing the sort and then editing/deleting a card keeps the selected sort
- **Works with all filters**: sort applies after tag filter and Struggling filter are applied
- **Grouping preserved**: cards remain grouped by book; sort applies within each group
- **Default "Newest"**: matches natural expectation of seeing recently-created cards first
- **"Most struggling"**: surfaces cards where `hardCount >= 1`, ordered by most hard answers first — distinct from the binary "Struggling filter" (N41) which shows/hides cards with `hardCount >= 3`
