## Feature details

### Quick-add widget
- Always visible at top of the annotation sidebar
- Single "✏️ Add note…" dashed-border button; clicking it hides the button and shows a textarea + Save/Cancel row
- On Cancel: form hides, button reappears, textarea cleared
- On Save (empty): `showToast('Note cannot be empty', 'error')` — no write
- On Save (with text): annotation saved, sidebar re-renders, trigger button reappears

### Freeform annotation object
```json
{
  "id": "ann_<timestamp>_<rand>",
  "chapterId": "<ctx.chapterId>",
  "bookId": "<ctx.bookId>",
  "start": <window.scrollY>,
  "end": <window.scrollY>,
  "color": "freeform",
  "text": "",
  "note": "<user text>",
  "type": "freeform",
  "createdAt": "<ISO string>"
}
```

### Chip appearance
- Purple left-border (`#a78bfa`)
- `✏️ NOTE` label in small uppercase purple text
- Note text as `.ann-chip-text`
- Date + Delete button in the footer row
- Click scrolls page to `ann.start` (the scroll Y captured at note creation)

### Book Notes panel
- Freeform notes render with the same purple chip style
- No ＋ Card button (note has no highlight text for the front)
- Clicking navigates to the chapter (same as other chips)
