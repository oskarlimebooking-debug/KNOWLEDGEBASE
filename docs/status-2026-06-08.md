## Feature details

### N34 — Card preview popover

- Single `#deck-card-preview` div (position:fixed, z-index:9200, pointer-events:none)
- Populated on each `mouseenter` with card's `front`, `back`, `tags`, meta
- Positioned 18px right of cursor; flips left if it would overflow `window.innerWidth - 16`; clamped within vertical viewport
- `@keyframes dcpFadeIn` gives a subtle 100ms slide-in from 4px above
- Not shown in bulk-select mode (`_deckSelectMode` guard)
- Confirmed working via browser eval: `previewVisible: "block"`, front/back/meta all populated correctly

### N35 — Freeform note delete button

- `✕` button sits in `.ann-chip-footer` after the date label, aligned right via `margin-left: auto`
- Hover state turns it red (`var(--danger, #ef4444)`)
- `stopPropagation()` prevents the click from also triggering navigation to the chapter
