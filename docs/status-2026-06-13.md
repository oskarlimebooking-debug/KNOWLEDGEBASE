## Feature details

### N41 — Card difficulty stats

- **Chip format**: `H{n}` (red), `G{n}` (yellow-amber), `E{n}` (green) — compact, clearly color-coded
- **Only shown when data exists**: Chips only render if `hardCount + goodCount + easyCount > 0`; new cards with no reviews show nothing
- **Dark theme**: Separate CSS rules with `rgba()` backgrounds and lighter text colors
- **Struggling button**: Shown only when `all.filter(c => c.hardCount >= 3).length > 0`; shows count in parentheses
- **Filter behavior**: Toggle — click once to activate (button fills red), click again to deactivate
- **Empty state for filter**: Shows "No cards match the Struggling filter." message
- **sm2Update persistence**: Counts accumulate across reviews; never reset (even when the SM-2 interval resets on Hard)
- **IDB changes**: No schema change — `hardCount`, `goodCount`, `easyCount` are new optional fields on `review_cards` objects (undefined = 0 for existing cards)
