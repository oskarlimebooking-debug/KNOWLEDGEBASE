### Score persistence

Each mode persists best scores in the `generated` IDB store:

| Mode | Store key |
|---|---|
| Classic | `quiz_scores_<chapterId>` (existing) |
| Speed Round | `speed_scores_<chapterId>` |
| Fill in the Blanks | `fill_blanks_scores_<chapterId>` |
| Connections | `connections_scores_<chapterId>` |

Best scores are displayed on the Quiz Hub cards on next visit.
