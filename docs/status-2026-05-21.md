## Feature details

### Phase N16 — CSV Card Import

**UI flow:**
1. Open Review Deck (via SRS chip or review session → Manage Deck)
2. Click "📥 CSV" button in the header
3. File picker opens — select any `.csv` or `.txt` file
4. Cards import instantly; toast shows count

**CSV format accepted:**
```
front,back
What is photosynthesis?,The process by which plants convert light to energy
"What is osmosis?","Movement of water across a semipermeable membrane"
```

**Parser details (`_parseCsv`):**
- Handles `"quoted fields"` with embedded commas
- Handles `""` escaped double-quotes inside quoted fields
- Normalises `\r\n` and `\r` line endings
- Skips blank lines

**Header auto-detection:**
- If the first row's first cell matches `/^(front|question|term|word|prompt)$/i`, that row is skipped

**Deduplication (`_csvIdFromFront`):**
- ID = `csv_` + djb2 hash of first 80 chars of the front text
- Importing the same CSV twice silently skips existing cards and reports the duplicate count

**Card defaults on import:**
- `ease: 2.5`, `interval: 0`, `repetitions: 0`, `nextReview: today`
- `bookId: 'csv_import'`, `chapterId: 'csv_import'` (grouped as "Unknown book" in deck manager)
