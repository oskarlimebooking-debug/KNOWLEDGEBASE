### Phase F — Lazy GIS loader (F11)

- `_loadGIS()`: injects `https://accounts.google.com/gsi/client` on first call, caches
  promise in `_gisLoadPromise`. No-op on subsequent calls. Error sets promise to null so
  next call retries.
- Called from `connectGoogleDrive()` before first tap, and in background from
  `_initDriveState()` when a client ID is stored (pre-warms for the next Connect tap).

### Phase F — OAuth token client (F1)

- `_initTokenClient(clientId)`: initialises `google.accounts.oauth2.initTokenClient` with
  the user-supplied client ID and `drive.appdata` scope. Callback stores token +
  expiry in `syncState` and settings.
- `_ensureToken()`: returns cached token if not within 60 s of expiry; throws `'401'`
  otherwise so callers can call `_handleExpiredToken()`.

### Phase F — Connect / Disconnect (F1)

- `connectGoogleDrive()`:
  1. Reads `#setting-google-client-id`, validates non-empty, saves to settings.
  2. If GIS not yet loaded: shows toast "Loading… tap Connect again", fires `_loadGIS()`.
  3. On second tap: calls `_initTokenClient`, then `_tokenClient.requestAccessToken()`
     synchronously (must be inside a user gesture — no `await` before this call).
  4. Awaits promise resolved by the GIS callback.
  5. On success: calls `_findOrCreateSyncFile()`, sets status → `connected`.
- `disconnectGoogleDrive()`: confirms, POSTs to the token revoke endpoint, clears all
  stored tokens and file ID, resets UI.

### Phase F — Find or create sync file (F2)

- `_findOrCreateSyncFile()`: `GET /files?spaces=appDataFolder&q=name='headway-sync.json'`.
  If found, stores `files[0].id`. If not found, creates via `POST /upload/drive/v3/files?
  uploadType=multipart` with `metadata: { name, parents: ['appDataFolder'] }` and
  `file: '{}'`. Stores resulting file ID in `syncState.syncFileId` and settings.

### Phase F — getSyncDataForUpload (F4)

- Reads all five IDB stores in parallel via `Promise.all`.
- Builds settings snapshot from the raw `settings` store (by key), whitelisting defined
  keys + all `prompt_*` keys.
- `excludePdfData = true` (used by `syncNow` to save memory): drops `pdfData`, sets
  `_pdfDataExcluded: true`.
- `excludePdfData = false`: converts `ArrayBuffer` pdfData to base64 via
  `_arrayBufferToBase64` (8 KB chunk loop — safe on iOS).

### Phase F — uploadToCloud (F5)

- Confirms before overwriting cloud. Calls `getSyncDataForUpload(false)` (full snapshot
  including PDF bytes). `JSON.stringify`s the whole object and PATCHes via
  `_patchSyncFile(jsonStr, token)` with a 120 s `AbortController` timeout.

### Phase F — downloadFromCloud (F6)

- Confirms ("This will OVERWRITE your local library"). Fetches cloud JSON via
  `_fetchCloudData()` (60 s timeout). Calls `_importSyncData(data, false)` which writes
  all stores and restores settings. Re-renders library.

### Phase F — syncNow — smart merge (F7)

Seven-stage sequence:
1. Download cloud snapshot (60 s timeout; 401 → `_handleExpiredToken`).
2. `getSyncDataForUpload(true)` — local snapshot with no PDF bytes (memory safety).
3. `_mergeData(local, cloud)` — last-write-wins by date field.
4. `_importSyncData(merged, true)` — writes IDB; `_pdfDataExcluded` entries: restores
   local PDF from IDB before writing, so no PDF data is lost.
5. `_streamUpload(merged)` — reads books fresh from IDB, inflates PDF to base64 chunk by
   chunk, builds a `Blob` from string parts (never materialises a single 200 MB string),
   PATCHes with 120 s timeout.
6. Re-renders library, updates `lastSyncAt`.

### Phase F — Merge algorithm (F8)

- `_mergeData(local, cloud)`: merges each store array via `_mergeArrayById`.
  Settings: `Object.assign({}, cloud.settings, local.settings)` — local device wins.
- `_mergeArrayById(localArr, cloudArr, dateField)`:
  - Seeds map from cloud items.
  - For each local item: if `dateField` set and item exists in map, only overwrites if
    `local[dateField] >= cloud[dateField]`. Otherwise always overwrites.
  - Returns `[...map.values()]`.
- Date fields: `books → addedAt`, `progress → date`, `generated → generatedAt`,
  `chapters → null` (latest import wins).

### Phase F — _importSyncData

- Books: if `_pdfDataExcluded && preserveLocalPdf`, fetches existing local book and
  restores its `pdfData` before writing. If `_pdfDataIsBase64`, inflates base64 string to
  `ArrayBuffer` via `atob` + `Uint8Array`.
- Chapters / progress / generated: bulk `dbPut`.
- Settings: `setSetting(key, value)` for each whitelisted entry from cloud.

### Phase F — Token expiry handling (F10)

- `_handleExpiredToken()`: clears `syncState.accessToken`, updates status to
  `'error'`, shows toast "Google session expired. Open Settings → Connect again.",
  calls `_renderSyncUI()` to hide action buttons.
- All sync functions catch `'401'` error string and route to this handler.
