## Feature details

### N37 — Chapter word count badge

- **Location**: Right side of each `.chapter-list li` row (after feed indicator)
- **Data source**: `c.wordCount` (stored at import time, line ~2434/2536 in `index.html`); fallback: `c.text.split(/\s+/).filter(Boolean).length`
- **Format**: `100 words` for counts < 1 000; `3.2k words` for counts ≥ 1 000
- **Omitted**: if computed count is 0 (avoids "0 words" for empty/stub chapters)
- **Style**: 11px, dimmed, no-wrap, flex-shrink 0 — stays on the right edge without pushing title
