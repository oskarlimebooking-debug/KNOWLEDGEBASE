## Feature details

### Phase T3 тАФ Reading time estimate

Progress line format:
```
Ch 3 of 5 ┬╖ 40% done ┬╖ ~1 min
```
- Reads `readingWpm` setting (default 250)
- Uses `chapter.wordCount` (stored at import time)
- `Math.max(1, Math.round(wordCount / wpm))` тАФ always at least 1 min
- Omitted silently if `wordCount` is falsy (very old imported chapters)

### Phase N17 тАФ Chapter bookmarks

**UI flow:**
1. Click ЁЯФЦ in chapter header
2. Click "+ Add bookmark here" тЖТ saves `{scrollY, label: "Chapter Title тАФ XX%"}` to IDB
3. Dropdown re-opens shows the saved bookmark with label
4. Click label тЖТ smooth scroll to that position, menu closes
5. Click тЬХ тЖТ deletes from IDB, menu updates in-place

**IDB schema:**
```
bookmarks: { id (UUID), bookId, chapterId, scrollY, label, createdAt }
  index: chapterId
  index: bookId
```

**Bookmark label format:** `"Chapter Title тАФ 47%"` (percent of scroll height)

### Phase N18 тАФ CSV card export

**Output format:**
```csv
front,back
"What is photosynthesis?","The process by which plants convert light to energy"
"What is osmosis?","Movement of water across a semipermeable membrane"
```
- All fields double-quoted
- Internal `"` chars escaped as `""`
- Filename: `headway-deck-YYYY-MM-DD.csv`
