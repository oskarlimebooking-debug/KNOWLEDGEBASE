## Feature details

### Entry point

⬇ icon button in the chapter header, between the 🖊 annotations button and the ✨ format button.

### Dropdown menu

```
⬇ [button clicked]
┌─────────────────────────────┐
│ Copy as Markdown            │
│ Headings, bold, links…      │
├─────────────────────────────┤
│ Copy as Plaintext           │
│ Clean text, no markup       │
└─────────────────────────────┘
```

Positioned with `position: fixed; z-index: 3000`, aligned to the button's right edge. Dismissed by clicking outside.

### Copy as Markdown

Output:
```
# Chapter Title

## Heading 1

Body with **bold** and *italic* preserved.

- bullet items
```

### Copy as Plaintext

Output:
```
Chapter Title

Heading 1

Body with bold and italic stripped.

bullet items (no dashes)
```

### Error handling

If `App._chapterCtx` is null (no chapter open), shows `'No chapter open'` error toast. If `navigator.clipboard.writeText` throws (rare, non-HTTPS non-localhost), shows `'Clipboard write failed — check browser permissions'`.
