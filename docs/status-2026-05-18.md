## Feature details

### Entry points

Three ways to open the deck manager:
1. **"🃏 N in deck" stat chip** on the library home screen — always visible when the deck has any cards. Shows total card count (not just due count).
2. **"Manage Deck" button** in the review session completion summary.
3. **Escape key** closes the overlay when open.

### Deck manager overlay

Full-screen overlay (z-index 2100, above the review session overlay) showing:

```
🃏 Review Deck    [Search cards…]    ✕
─────────────────────────────────────
3 cards total · 2 due today

UNKNOWN BOOK
┌─────────────────────────────── ✏️ 🗑️ ┐
│ What is spaced repetition?           │
│ A learning technique where…          │
│ Due today  Interval: 1d  Ease: 2.5  Reps: 1 │
└───────────────────────────────────────┘
┌─────────────────────────────── ✏️ 🗑️ ┐
│ What is ease factor?                 │
│ A multiplier in SM-2…                │
│ Next: 2026-05-25  Interval: 14d  Ease: 2.8  Reps: 3 │
└───────────────────────────────────────┘
```

### Card display

- **Front**: bold, truncates naturally, right-padded to clear action buttons
- **Back**: dimmed text below front
- **Meta row**: "Due today" (accent colour + bold) or "Next: YYYY-MM-DD", plus Interval, Ease, Reps
- **Actions**: ✏️ edit, 🗑️ delete (top-right corner)

### Edit in place

Clicking ✏️ expands the card inline with:
- "FRONT" and "BACK" labelled textareas pre-filled with current text
- "Cancel" dismisses the form (no change)
- "Save" writes updated front/back to IDB and refreshes the view
- Clicking ✏️ again while form is open collapses it

### Delete

Clicking 🗑️ shows a browser `confirm()` dialog. On confirmation, deletes the card from IDB via `dbDelete('review_cards', id)` and re-renders the group.

### Search

Live-filter search input in the header. Filters across front AND back text case-insensitively. Shows "No cards match your search." when empty. No debounce needed (instant, pure JS filter on the already-loaded array).

### Grouping

Cards grouped by book. Book titles resolved from `libraryCache`. Cards for books not in cache show "Unknown book". Cards sorted: due today first, then by `nextReview` ascending.
