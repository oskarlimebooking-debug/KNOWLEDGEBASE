## Feature details

### Color stats bar

Appears **above the chapter-grouped chips** in the Notes tab when the book has ≥ 1 annotation.

```
● 12   ● 5   ● 3   ● 1   ● 2
```

- Dots are inline-styled with the exact border colors already used by the chip variants (`#eab308`, `#ec4899`, `#22c55e`, `#0ea5e9`, `#a78bfa`).
- Only colors with count > 0 are shown; the row is omitted entirely on zero annotations (the empty-state message shows instead).
- Counts include both highlight chips and freeform notes (purple dot).
- The row is a flex container that wraps gracefully on narrow viewports.
