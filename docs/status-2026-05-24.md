## Feature details

### Phase N20 — Book notes panel

**UI flow:**
1. Open any book → see "Chapters" (active) | "Notes" tab bar below the hero
2. Click "Notes" → chapters panel hides; notes panel renders asynchronously
3. Notes panel shows all annotations grouped under their chapter (CHAPTER N: TITLE headers)
4. Each annotation chip shows: highlighted text, italic note (if any), date, "＋ Card" / "✓ In deck" button
5. Click a chip → navigates to that chapter (`openChapter(bookId, chapterId)`)
6. "＋ Card" button: same two-path logic as sidebar — with note → one-click; without note → inline textarea quick-form
7. Notes tab label updates to "Notes (N)" with the total annotation count
8. Re-opening Notes tab re-renders with fresh data (no stale cache)

**Reuse:** All chip styles (`.ann-chip`, `.ann-chip-yellow/pink/green/blue`, `.ann-chip-text`, `.ann-chip-note`, `.ann-chip-footer`, `.ann-chip-meta`, `.ann-chip-card-btn`, `.ann-card-quick-form`) are reused from the N2/N19 sidebar — no style duplication.
