## Feature details

### Phase T6 — Keyboard shortcuts modal

**UI:**
- `?` button in the library header (between 🔭 Discover and ⚙ Settings)
- Global `?` keypress (when not focused on input) toggles modal
- `Esc` closes the modal
- Backdrop click closes the modal
- `×` close button in modal header

**Sections documented:**
1. **Global** — `⌘K` search, `?` shortcuts, `Esc` dismiss
2. **Search Palette** — `↑`/`↓` navigate, `Enter` open
3. **Chapter Reading** — `←`/`→` chapters, `A` annotations, `B` bookmarks, double-click define, select-text annotate
4. **Spaced Repetition Review** — `1` Hard, `2` Good, `3` Easy

**New shortcuts wired (previously button-only):**
- `←` / `→` — previous/next chapter (chapter view, not in input)
- `A` — toggle annotation sidebar (chapter view, not in input)
- `B` — open bookmark dropdown (chapter view, not in input)
- `1` / `2` / `3` — SRS Hard/Good/Easy (active review session only)
